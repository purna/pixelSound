Google Sheets Integration

Link a Google Sheets spreadsheet to manage ratings, tags, and descriptions for all audio files

Sheet Structure:

  • Column A: File Path (Google Drive URL or local path)
  • Column B: Display Name
  • Column C: Tags (comma-separated)
  • Column D: Description
  • Column E: Rating (1-5)
  • Column F: Folder Collection (library name)

Note: The first row should contain column headers. The sheet must be published to web (File → Share → Publish to web).
